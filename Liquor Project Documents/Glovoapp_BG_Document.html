Points to be taken care before running the script:

  1. Change month date:
    • Go to the db_config.py file and change the month_date variable value according to "DayMonthYear" format.
    • Date will be fixed "08"
    • For example: month=11, date: 08, year=2024 then month_date = '08112024'
  1. Page Save file drive available in PC (for example: D) and check the file path as mentioned in db_config.py file for page save & check EXPORT.py for excel output file

Steps for running the script for data extraction:

  • Run the link_revised.py file to extract links and data into the table as db_data_table name provide in db_config.py file.
  • After running the link_revised.py, the database and table will be created as per the details provided in db_config.py if database and table do not exist in given connection.
  • In this project, no new old link concept exists so only one table is created for getting the links and extracting data.
  • While running the script, please check if pages are properly saved

Checking the status during script execution:

In SQL, execute the following query to check the count of inserted data (replace with your date):

SELECT COUNT(*) FROM `spirit_data_glovoap_billa_08102024`

Running further processes:

Just run the script. File generation code is already set to run when the process is done and when the spider closes. The excel file will be available on the given path.

Points to note for file generation:

  • An attempt has been made to convert the file into English language, but don't depend on this translation. Make sure to verify from your side. (It's not working properly for large data)
  • If the file is not generated after completion of the process, run EXPORT.py file manually. If facing any issues, contact the developer.
  • You will need to manually translate from native language to English. (Using Google Sheet)
