Points to be take care before running the script:

  1. Change month date:
    • Go to the db_config.py file and change the month_date variable value according to "DayMonthYear" format.
    • Date will be fixed "08"
    • For example: month=11, date: 08, year=2024 then month_date = '08112024'
  1. Page Save file drive available in PC (for example: D) and check the file path as mentioned in db_config.py file for page save & check EXPORT.py for excel output file.
  1. Ensure that no link table exists before running py. There is no need for old or new concepts; simply run the link_extractor.py script, which will generate the links table each time.

Steps for running the script for data extraction:

  1. Run the link_extractor.py file to extract links into the table as db_links_table name provided in db_config.py file.
  1. Check the count of all inserted links using the mentioned query below.
  1. After running the link_extractor.py, run page_save.py for page save.
  1. After the table is generated, saving the page is required using Selenium. There are two scripts available for saving the page: one uses threading, and the other does not. If you want to save the page faster, use the threading script. However, if you're experiencing response issues, use the normal page-saving script.
  1. While running page_save.py you can check page save status using the SQL queries mentioned below.
  1. Once page save is done, run data_extractor.py.
  1. The data_extractor.py script retrieves links from the spirit_links table where the status is set to "SAVE" and processes them for data extraction.
  1. In this project, new links are added and old links are also checked for data extraction.
  1. For checking how many new links are added, please refer to the SQL queries mentioned below.
  1. While running the script, please check if pages are properly saved.

Checking the status during script execution:

In SQL execute query:

  • For checking the count of links added in spirit_links table:
    SELECT COUNT(*) FROM `spirit_links`
  • For checking page save count of links from spirit_links table:
    SELECT COUNT(*) FROM `spirit_links` WHERE status="SAVE"
  • For checking the status of extracted data (Pending, Done, or Not Found):
    SELECT "Total", COUNT(*) FROM `spirit_links` UNION
SELECT "Done", COUNT(*) FROM `spirit_links` WHERE STATUS="Done" UNION
SELECT "Pending", COUNT(*) FROM `spirit_links` WHERE STATUS="Pending" UNION
SELECT "SAVE", COUNT(*) FROM `spirit_links` WHERE STATUS="SAVE" UNION
SELECT "Not Found", COUNT(*) FROM `spirit_links` WHERE STATUS="Not Found"

Running further process:

  1. First, run the file for link extraction.
  1. After that, run the data page save file.
  1. Finally, run the extraction file.
